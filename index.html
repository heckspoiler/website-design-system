Windows 95 Color Scheme

Color Scheme

In this section we'll take a closer look at the Windows 95 color scheme.

Color Name: Teal
HEX: #008080
RGB: rgb(0,128,128)
Usage: mainly background

Color Name: Unknown
HEX: #C0C0C0
RGB: rgb(192,192,192)
Usage: background for boxes

Color Name: Lightgrey
HEX: #D3D3D3
RGB: rgb(211,211,211)
Usage: hover state for buttons

Color Name: Gradient
Usage: Upper Box
Code:
background: -webkit-linear-gradient
(left,#08216b,#a5cef7)

Color Name: Blue
HEX: #08216b
RGB: rgb(0,0,255)
Usage: hover state links, selection background

Color Name: Black
HEX: #000000
RGB: (0, 0, 0)
Usage: text

Color Name: White
HEX: #FFFFFF
RGB: rgb(255,255,255)
Usage: selection text, text for upper box
